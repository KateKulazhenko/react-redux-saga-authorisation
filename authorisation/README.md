## Requirements

* Node >= 9.3.0
* NPM >= 5.6.0

You can use other versions, but the project is build on the use of these versions.

## Installation

```sh
git clone https://github.com/KateKulazhenko/react-redux-saga-authorisation.git

npm install
```

## Start

```sh
npm start
